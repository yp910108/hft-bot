## 策略核心逻辑

### 一句话

**开场双边建仓，持续 12 分钟高频低买高卖循环，通过逐笔止盈回收现金压低净持仓成本。最终净持仓 sum_avg 中位 0.94 < 1，无论哪方赢大概率盈利。**

### 操作生命周期

```
┌─ 开场（开盘后 9 秒入场）─────────────────────────────────────────┐
│ 快速双边建仓，前 60s 投入约 $55（UP $25 + DN $23，中位）          │
│ 首笔 73% 买便宜侧（price < 0.50）                                │
│ 51% 的场次前 60s 内已有卖出操作（快速周转）                       │
└───────────────────────────────────────────────────────────────────┘
       ↓
┌─ 持续 ~12 分钟（中位操作跨度 754s）────────────────────────────────┐
│ 中位间隔 2 秒，37% 操作在同一秒内完成（批量成交）                  │
│                                                                    │
│ 核心循环（逐笔止盈止损）：                                         │
│   买入 → 挂 Maker 卖单 → 涨了 5 分成交止盈 / 跌了 4 分割肉止损   │
│   LIFO 特征：最近买入的那笔，中位持有 24 秒就卖出                  │
│   盈利率 78%，止盈中位 +0.05，止损中位 -0.02                       │
│                                                                    │
│ 四时段卖出占比：23% → 34% → 40% → 46%（越后面越"收割"）          │
│ 止盈目标随时间放大：Q1 +0.07 → Q4 +0.12                           │
└───────────────────────────────────────────────────────────────────┘
       ↓
┌─ 收手（末笔距结算中位 165 秒）────────────────────────────────────┐
│ 不是一刀切停手，而是自然收敛                                       │
│ 后期买入减少、卖出增加（把仓位变现）                               │
│ 最终保留净持仓等结算：UP 卖出 47%、DN 卖出 44% 的买入量           │

```

## 现金流与资金效率

这是理解策略的关键。他不是"投入 $778 等结算"，而是**高频买卖循环回收现金**：

| 指标 | P50 | 均值 |
|------|-----|------|
| 总买入花费 | $500 | $591 |
| 总卖出回收 | $287 | $351 |
| **净投入（真实占用资金）** | **$205** | **$240** |
| 回收率 | 57% | 58% |

典型盈利场：买入 $373、卖出回收 $302（回收率 81%），净投入仅 $71，赚 $10.68（净回报 15%）。

**卖出不是亏损，是策略的利润收割手段。** 每次止盈卖出都回收了现金 + 锁定了利润差价，使得净持仓的等效买入价被大幅压低。

## 详细决策规则

### 1. 买入触发

| 维度 | 数据 | 结论 |
|------|------|------|
| 入场时机 | 开盘后中位 9 秒 | **极快入场，不观望** |
| 首笔方向 | UP 220 / DN 182 | 不挑方向，哪边先到就先买 |
| 首笔价格 | 73% < 0.50 | **优先买便宜侧** |
| 前两笔同侧 | 82%（330/402）| 连续同侧铺底仓 |
| 同侧连续买入价格变化 | 37% 下跌后买 / 26% 持平 / 37% 上涨后买 | **不区分涨跌，持续买入** |
| 单笔买入量 | P50=8.5 股，$4.0 | 小单高频 |
| 连续买入 streak | P50=2, P75=5, Max=41 | 通常 2~5 笔后穿插卖出 |

**买入逻辑推断**：
- 开场即双边建仓，不预测方向
- 持续在两侧挂 Maker 买单，便宜了就买（价格给机会就接）
- 不刻意等下跌才买，而是持续有在场（维持双边流动性提供者角色）
- 单笔小量（$4），分散风险

### 2. 卖出触发（止盈止损）

**止盈止损随时间变化**：

|------|----------|----------|--------|

**卖出逻辑推断**：
- **不是看绝对价格，而是看每笔买入的相对利润**
- 每笔买入独立跟踪，涨了 5 分左右就挂 Maker 卖单等成交
- Maker 卖出（87%）= 提前在高位挂卖单等人吃，零费率
- 持有 24 秒（中位）= 买入后很快就挂好了卖单
- 如果跌了，快速割肉（止损中位 -0.02），不恋战

### 3. 库存管理（持仓不平衡时的行为）

| 不平衡度 | 买入轻侧 | 买入重侧 | 卖出重侧 | 卖出轻侧 | 再平衡行为 |
|----------|----------|----------|----------|----------|-----------|
| 低(<0.2) | 29.8% | 28.3% | 21.2% | 20.6% | 51.1% |

**关键发现**：
- 不平衡度高时，再平衡行为仅 53%（接近随机）
- **不刻意对冲**——对称度是低买高卖循环的自然副产品
- 双边对称度演变：Q1=0.55 → Q2=0.77 → Q3=0.82 → Q4=0.85（自然收敛）
- 高不平衡时甚至有 39% 在买入重侧（继续加仓涨的那边），说明他跟价格走不跟平衡度

## Maker / Taker 拆解

data-api 的 `/trades` 用 `takerOnly` 参数反推：

| 类型 | Maker | Taker |
|------|-------|-------|
| **卖出** | **87%** | 13% |
| **买入** | **78%** | 22% |

**对策略复制的意义**：
- Maker 费率 0%，Taker ~4%。止盈中位只有 +0.05，如果卖出走 Taker 扣 4% 后利润几乎归零。
- 87% 卖出是 Maker = **提前在高位挂好卖单等人来吃**，不是看到涨了再追着 Taker 卖。
- 这意味着策略的核心节奏是：买入成交 → 立刻在更高价挂 Maker 卖单 → 等市场来吃。
- 如果市场不来吃（没涨到卖单价），那笔单就一直挂着或最终止损。

**盈利机制**：
```

                                              ↓
                                 UP 净持仓成本被压低（或者卖掉了，成本归零）
                                              ↓
                         同时 DN 也在做同样的事 → 双边净成本都被压低
                                              ↓
                           最终 sum_avg < 1 → 结算时无论谁赢都赚
```

| 模式 | 场数 | 占比 | 说明 |

**第三类（sum_avg<1 却亏）的原因**：
- 虽然双边均价之和 <1，但赢家侧净持仓/输家侧净持仓 中位只有 **0.84**

## 策略本质总结

0xe00740bc 是一个**高频做市商库存管理策略**：

1. **双边持续挂 Maker 单**：买卖两头都用 Maker（0 费率），在盘口两侧持续提供流动性
2. **逐笔微利止盈**：每笔买入独立跟踪，涨 5 分挂 Maker 卖单，成交即锁利润
3. **快速止损**：跌了就割（中位 -0.02），不扛单
4. **高频周转压成本**：57% 的买入资金被卖出循环回收，净持仓成本远低于表面买入价
5. **不刻意对冲**：对称度是循环的副产品，不是显式目标
6. **靠波动赚钱**：需要价格运动来触发止盈，低波动场是天然短板

## 复制要点

### 必须具备的能力
1. **Maker 挂单能力**：买卖都挂 Maker（0 费率），尤其卖出 87% 是 Maker
2. **逐笔止盈跟踪**：每笔买入独立跟踪利润，涨到阈值立刻挂卖单
3. **高频操作**：中位间隔 2 秒，需要低延迟 WebSocket + 快速决策
4. **双边同时建仓**：开场 9 秒入场，两侧同时铺单
5. **动态止盈目标**：Q1 +0.07 → Q4 +0.12，随场内进度调整

### 关键参数（从数据提取）
- 入场延迟：开盘后 ≤9 秒
- 单笔大小：8~14 股（$4~6）
- 止盈目标：+0.05~0.12（随时间递增）
- 止损阈值：-0.02~-0.07（随时间放宽）
- 卖出方式：Maker 挂单（买入成交后立刻在高位挂卖单）
- 收手：不硬停，自然收敛到距结算 ~2.5min

### 风险
- **低波动场必亏**：波动<0.4 胜率仅 23%——需要波动过滤器或减仓策略
- **Maker 成交率**：如果卖单挂了没人吃，止盈就失败了，最终 sum_avg 压不下来
- **流动性**：118 笔/场需要盘口有足够深度承接
- **竞争**：如果多个做市商用同样策略，利润空间被压缩
